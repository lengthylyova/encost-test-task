## Как использовать?
* <u>Клонируйте</u> этот репозиторий;
```bash
git clone https://github.com/lengthylyova/encost-test-task.git
```
* Запустите файл app.py;
```bash
python <some>/<path>/<to>/encost-test-task/app.py
```
